# Governance

## Decision-Making

tang-edge follows a single-maintainer model. All technical decisions, code reviews, merges, and releases are made by [@tang-edge-worker](https://github.com/tang-edge-worker).

Community input is welcome via [GitHub Issues](https://github.com/tang-edge/tang-edge/issues) and [Pull Requests](https://github.com/tang-edge/tang-edge/pulls).

## Roles

| Role | Responsibilities |
|------|-----------------|
| **Maintainer** | Merge PRs, cut releases, triage issues, security response |
| **Contributor** | Submit PRs, report bugs, suggest features |

## Continuity

This project is licensed under GPL-3.0. If the maintainer becomes unavailable, the community is free to fork and continue development under the same license.
